# java-web-maven-spring-thymeleaf-hello

## Description
A POC for springboot web app with thymeleaf support.

## Tech stack
- java
- maven
  - springboot
  - thymeleaf
  - bootstrap

## Docker stack
- openjdk:11-jdk

## To run
`sudo ./install.sh -u`
Available at http://localhost

## To stop
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
